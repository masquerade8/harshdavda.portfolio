Crypto Threat Intelligence Platform
+

Scammers can clone a crypto exchange in under an hour, while security teams often discover the breach days later. This platform continuously monitors certificate transparency logs and OSINT phishing feeds, detecting copycat domains the moment they are registered - then scores them using fuzzy matching, homoglyph detection, domain age, and hosting reputation signals.

Once a threat is confirmed, the platform maps it to the appropriate regulator (VARA, SAMA, CBB, MAS), calculates the compliance deadline (as tight as 2 hours in GCC jurisdictions), and auto-drafts the notification email with all supporting evidence - full detection-to-compliance-draft automation.

Hardware Security · Embedded Systems · Authentication
Secure RFID Authentication System (Arduino Uno)
+

A secure RFID-based access control system using Arduino Uno and MFRC522 module, implementing a challenge-response authentication scheme based on XOR combined with UID-based validation - demonstrating applied embedded security principles at the hardware level.

Arduino Uno MFRC522 Challenge-Response Auth XOR Cryptography I2C LCD Embedded C
  • Verifies card UID against an authorized list before proceeding to cryptographic challenge
  • Reads a secret key stored securely in sector 1, block 4 of the RFID tag
  • Arduino generates a 4-byte random challenge, XORs with the card's key to produce the expected response
  • Access granted only when generated response matches expected - prevents replay and cloned-card attacks
  • LCD displays contextual feedback: Scan Card / Invalid Card / Access Granted
